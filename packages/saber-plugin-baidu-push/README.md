# saber-plugin-baidu-push

Baidu Push SEO plugin for [Saber](https://saber.land/).

## Install

```bash
yarn add saber-plugin-baidu-push
```

## Usage

In your `saber-config.yml`:

```yml
plugins:
  - resolve: saber-plugin-baidu-push
```
